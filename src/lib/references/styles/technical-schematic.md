# technical-schematic

Blueprint-grade technical diagrams with engineering precision, clean geometry, and systematic data presentation. Think architectural blueprints, NASA mission plans, or patent drawings.

## Color Palette

- **Blueprint variant**: Deep blue background (#1E3A5F → #0F2640) with white/light blue line work, cyan highlights (#00D4FF), amber callouts (#F59E0B)
- **Light variant**: White/light gray background (#F8F9FA) with dark blue line work (#2563EB), subtle gray grid, amber highlights
- **Lines**: Consistent stroke weights — 1px for grid, 2px for elements, 3px for emphasis
- **Data colors**: Blue (#2563EB), teal (#0891B2), amber (#F59E0B), red (#DC2626) for charts and indicators
- **Annotation**: White or amber text on dark backgrounds, dark blue text on light backgrounds

## Illustration Technique

### Technical Drawing
- All elements rendered with geometric precision — straight lines, perfect circles, measured angles
- Grid pattern visible in the background (subtle, but present) — engineering graph paper or dot grid
- Dimension lines with measurement annotations where relevant
- Technical symbols: arrows with precise tips, corner marks, reference lines
- Cross-section views, exploded diagrams, and schematic representations

### System Diagrams
- Components rendered as clean geometric shapes with labels
- Connection lines with defined endpoints (dots, arrows, or terminals)
- Flow arrows showing direction of data, process, or energy
- Grouping boxes with dashed borders for subsystems
- Status indicators: filled/empty circles, traffic lights, checkmarks

### Data Visualization
- Charts with precise grid backgrounds and axis labels
- Data points plotted accurately with value annotations
- Clean bar charts, line graphs, and scatter plots
- Gauge dials with needle indicators and range markings
- Tables with alternating row highlighting and aligned columns

### Technical Annotations
- Callout lines pointing from labels to specific components
- Reference numbers in circles (①, ②, ③) linking to legend
- Measurement units clearly displayed
- Tolerance or range indicators where applicable

## Typography

- Technical stencil or clean monospace/sans-serif (JetBrains Mono, DIN, Roboto Mono aesthetic)
- ALL-CAPS for labels and section titles
- Measurement annotations in smaller type with unit labels
- Consistent text sizes: title (large), section headers (medium), labels (small), annotations (smallest)

## Composition

- Strict grid alignment — everything snaps to an underlying grid
- Generous margins with technical border frame (title block at bottom or top)
- Systematic spacing between elements
- Clean lines connecting related elements

## Best For

System architecture, engineering specs, product teardowns, network diagrams, technical workflows, API documentation, infrastructure maps, scientific methodology
