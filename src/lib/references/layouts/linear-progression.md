# linear-progression

A sequential journey showing steps, timeline events, or chronological progression. Each node is an illustrated milestone with its own scene.

## Structure

### The Progression Line
- A clear connecting line, path, or track running horizontally or vertically through the canvas
- Nodes/markers at each key point — numbered circles, flags, or milestone markers
- Clear START and END points with distinct visual treatment
- Directional flow indicators (arrows, gradient, perspective)
- The line itself should be visually interesting — not just a thin rule, but a styled path (rope, wire, road, river, timeline bar)

### Node Scenes (5-8 nodes)
Each node along the progression is an illustrated scene:
- **Numbered marker**: Step number or date prominently displayed
- **Scene illustration**: A specific drawn vignette showing what happens at this step — people doing things, objects, diagrams, charts
- **Title**: Bold heading for the step/event (max 5 words)
- **Description**: 2-3 concise lines of content
- **Labels**: Annotations pointing to specific elements in the scene

### Alternating Layout
- Nodes alternate above and below the line (horizontal) or left and right (vertical)
- This creates visual rhythm and prevents a monotonous row
- Each node's illustration occupies roughly equal space

## Text Placement

- Title: large, prominent at the very top
- Subtitle: one line describing the progression arc
- Node titles: bold, at each milestone point
- Node descriptions: concise text panels adjacent to illustrations
- Dates/numbers: prominently displayed at each node
- Labels: directly on illustrations

### Bottom Stats Bar
- Horizontal strip at the bottom with 4-6 key metrics
- Format: "LABEL | VALUE" with dividers
- Slightly contrasting background

## Composition Flow

1. Eye enters at the title (top)
2. Follows the progression line from start node to end node
3. Pauses at each milestone scene for details
4. Lands on stats bar for data summary

- Step-by-step tutorials and processes
- Historical timelines and evolution stories
- Project milestones and roadmaps
- Workflow documentation
- Discovery narratives and scientific progress
